# 🦋 Portfólio — Luana Maria

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro, apresentando projetos, formação e informações de contato.

---

## 🔗 Acesso

Basta abrir o arquivo `index.html` em qualquer navegador moderno — não é necessário servidor ou dependência externa.

## 🗂️ Seções

| Seção | Descrição |
|---|---|
| **Início** | Apresentação com nome, frase de efeito e links de contato |
| **Sobre Mim** | Texto de apresentação profissional |
| **Projetos** | Cards expansíveis com preview e link para cada projeto |
| **Cursos** | Grade com formações e certificações |

---

## ✨ Funcionalidades

- **Menu responsivo** com dropdown para mobile e destaque no link ativo
- **Animação de scroll** na seção "Sobre Mim" via Intersection Observer
- **Cards de projetos expansíveis** ao passar o mouse, revelando preview da aplicação
- **Header dinâmico** que se torna transparente quando a página está no topo
- **Canvas de partículas** de fundo na seção de Cursos
- **Efeito flutuante** na ilustração principal via `@keyframes`

---

## 🛠️ Tecnologias

- HTML5 semântico (com `aria-*`, `role`, `<nav>`, `<main>`, `<address>`, `<article>`, `<figure>`)
- CSS3 (Flexbox, Grid, Custom Properties, Media Queries, animações)
- JavaScript vanilla (Intersection Observer, manipulação de DOM)
- Google Fonts: **Bebas Neue**, **Anton**, **Roboto**

---

## 📱 Responsividade

O layout foi projetado com breakpoints para os seguintes cenários:

| Largura | Comportamento |
|---|---|
| `> 1250px` | Layout desktop completo |
| `≤ 1000px` | Ajustes de posicionamento da ilustração |
| `≤ 862px` | Menu colapsa em dropdown |
| `≤ 720px` | Título e imagem redimensionados |
| `≤ 420px` | Layout otimizado para celulares pequenos |
